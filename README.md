###AWS EC2:

`sudo yum update`

`chmod 400 key.pem`

NodeJS:
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.32.0/install.sh | bash
. ~/.nvm/nvm.sh
nvm install 4.4.5
node -v
npm -v
```

Git:
`sudo yum install git-all`
